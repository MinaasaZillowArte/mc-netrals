Rekrutmen Admin

Kami mencari Netralers berbakat untuk bergabung dengan tim dan membangun komunitas NTRL menjadi lebih baik!

Admin Arsitek

Bertanggung jawab atas estetika dan proyek pembangunan di dalam server.

Tugas:

  • Merancang dan memimpin proyek bangunan komunitas.
  • Menilai dan memberi masukan pada karya pemain.
  • Menjaga konsistensi tema arsitektur server.

Syarat:

  • Memiliki portfolio bangunan yang mengesankan.
  • Menguasai WorldEdit/VoxelSniper adalah nilai plus.
  • Kreatif, komunikatif, dan bisa bekerja dalam tim.

Admin Keamanan

Menjaga ketertiban dan memastikan lingkungan bermain yang adil dan aman.

Tugas:

  • Menangani laporan dari pemain dengan cepat.
  • Memantau server dari aktivitas mencurigakan (cheat/grief).
  • Menegakkan peraturan server secara konsisten.

Syarat:

  • Memahami command dasar admin (kick/ban/mute).
  • Tegas, adil, dan punya rekam jejak yang baik.
  • Bisa tetap tenang dalam situasi konflik.

Admin Event

Merancang dan menjalankan acara-acara seru untuk menghidupkan komunitas.

Tugas:

  • Membuat konsep event yang unik dan menarik.
  • Mempersiapkan arena dan mekanisme event.
  • Memandu jalannya event dari awal hingga akhir.

Syarat:

  • Penuh ide kreatif dan bersemangat.
  • Terorganisir dan mampu mengatur acara.
  • Pandai berkomunikasi dan membangkitkan hype.
